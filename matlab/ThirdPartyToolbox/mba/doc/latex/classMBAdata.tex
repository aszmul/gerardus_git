\subsection{Detailed Description}
Holds the scattered data. \doxyref{MBAdata}{p.}{classMBAdata} -\/ Holds scattered data for Multilevel B-\/spline approximation (and interpolation) for functional surfaces. Note that this is a (mathematical) lower level interface; thus there is no check for the validity of given scattered data or of arguments passed to member functions. See further documentation in \doxyref{MBA}{p.}{classMBA} and MBAadaptive
